Class FileSource 

Unity implementation of the FileSource class. It will use Unity's

coroutines to asynchronously fetch data from the network via HTTP or HTTPS requests.

Inheritance
System.Object
FileSource
Namespace:Mapbox.Unity
Assembly:Mapbox.Unity.dll
Syntax
public sealed class FileSource : object, IFileSource
Remarks
A FileSource is needed for each MonoBehaviour in the game doing network requests.
The code is compatible for exporting to Unity's WebPlayer.
